Ներածություն

Այս գրառումով բացում եմ Սիմֆոնի2-ի մասին հայալեզու գրառումների շարք, որտեղ մասնավորապես կլինեն պաշտոնական փաստաթղթերի թարգմանություններ (ազատ ոճով), ինչպես նաև հեղինակային նյութեր։ Բլոգի առաջնային առաքելությունն է տարածել Սիմֆոնին Հայաստանում։ Բլոգի նյութերը կհրապարակվեն symfony.am կայքում

Բլոգի նյութերը կհրապարակվեն նաև kode.am-ում: Ձեր բլոգում նյութը տեղադրելու համար կապվեք ինձ հետ tigran[@]symfony.am

Վիզուալ կերպով ծանոթանալու համար կարող եք բեռնել PowerPoint Presnetation ֆայլը

Սկիզբ՝ բառիս բուն իմաստով: Symfony2-ով սկսել է նոր դարաշրջան PHP Framework-ների համար: Այն փոխում է Framework-ների մասին պատկերացումները:

 Ի՞նչու Սիմֆոնի (Երկու բառով)

  1. Ժամանակը գնում է ոչ թե հեծանիվներ հորինելու վրա այլ հենց ծրագրի լոգիկայի վրա աշխատելու, որն ավելի արդյունավետ է
  2. Նորարար կառուցվածք, որը թույլ է տալիս արդյունավետորեն լուծել ցանկացած առաջդ դրված խնդիր
  3. Ծրագրի հեշտ սպասարկում, թարմացումներ
  4. Խիստ տարանջատված ծրագրային կոդեր՝ լոգիկայի, տվյալների բազայի հետ աշխատելու, շաբլոնիզացիայի և էջերի կառուցվածքի համար
  5. Ավտոմատ ստեղծվող ծրագրային կոդեր, որն օգնում է պարզ քայլերը միանգամից կատարելու և բուն լոգիկան գրելուն անցնելու համար, այսինքն Դուք ժամանակ չեք ծախսի սկսելու վրա
  6. Նշեմ նաև պաշտոնական կայքում գրված առավելությունները.
    • Վարկանիշ
    • Արտադրողականություն
    • Լայն տարածում(հատկապես արևմուտքում)
    • Ինովացիաներ
    • Փաստաթղթավորում
    • Համագործակցելիություն

Ինչ է պետք սկսելու համար

  1. HTTP սերվեր, օրինակ՝ Apache,Nginx,IIS
  2. Տվյալների բազա, օրինակ՝ MySQL կամ MongoDB
  3. PHP (> 5.3.8), խորհուրդ է տրվում 5.4.x տարբերակները
  4. SSH
  5. PHP APC Extension
  6. Համատեղելիությունը ստուգելու համար հավաքեք
    [bash]php app/check.php[/bash]
